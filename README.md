## Quick start

### Backend
```
cd backend
cp .env.example .env
npm install
npm run seed   # seed sample products
npm run dev    # start server (nodemon)
```

### Frontend
```
cd frontend
npm install
npm run dev
```

Frontend (Vite) typically runs at http://localhost:5173 and backend at http://localhost:4000
