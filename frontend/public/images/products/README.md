# Hướng dẫn lưu ảnh sản phẩm

Vui lòng lưu các ảnh sản phẩm với tên file như sau:

1. **solar-system.jpg** - Hệ sinh thái trí tuệ (ảnh 1 - hệ mặt trời)
2. **castle-blocks.jpg** - Kiến trúc sư nhỏ (ảnh 2 - lâu đài)
3. **counting-board.jpg** - Cây toán học (ảnh 3 - bảng đếm số)
4. **alphabet-tree.jpg** - Vườn ngôn ngữ (ảnh 4 - cây chữ cái)
5. **math-abacus.jpg** - Khám phá vũ trụ (ảnh 5 - bàn tính toán)

## Cách lưu:
1. Tải 5 ảnh sản phẩm từ chat
2. Đổi tên theo danh sách trên
3. Copy vào thư mục này: `frontend/public/images/products/`

Sau khi lưu xong, ảnh sẽ tự động hiển thị trên website!
